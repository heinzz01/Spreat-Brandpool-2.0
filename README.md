# Spreat · Markenpool 2.0 — klickbarer Prototyp

Interner Prototyp zur Abstimmung. **Kein Produktionscode**, keine echten Accounts:
Registrierung und Login sind simuliert, Daten liegen im Browser (localStorage).

## Einstieg

👉 **[Prototyp öffnen](https://heinzz01.github.io/Spreat-Brandpool-2.0/)**

## Wege durch den Prototyp

1. **Startseite** — Rollenauswahl: Einzelhändler oder Marke.
2. **Als Einzelhändler** → Brandpool zum Stöbern, Hinweis auf Registrierung,
   Werbefläche nach jeder 5. Marke → Registrierungsformular (Ladenfotos,
   Verkaufsfläche, Preis-Positionierung, No-Gos, Vorlieben, Offenheits-Status).
3. **Als Marke** → Registrierung (Website-Weg endet im Danke-Screen: Spreat baut das
   Profil auf) oder Login mit bestehendem Markenprofil → Markenprofil-Dashboard mit
   Händler-Karte, Empfehlungen und Bedarfs-Board.

Einmal angemeldet bleibt die Rolle gespeichert — „Abmelden" führt zurück zur Startseite.
